##  Key Insights (What the Data Revealed)

 **Strong churn drivers**
- Month-to-month contracts  
- High monthly charges  
- Early customer tenure (0â€“18 months)

 **High-risk segments**
- Senior citizens  
- Customers without value-added services  
- Paperless billing users with low loyalty

 **Service & operator patterns**
- Fiber-optic users churn more despite premium plans  
- Multiple-line users show better retention  
- Operator policies impact churn more than gender
